# lesson03
Lesson #03 (05 &amp; 06) | JS12@ORT Циклы в JavaScript. Универсальный циклы с условием while, do-while. Цикл со счётчиком for. Цикл перебора итерабельных объектов for-of. Цикл перебора свойств (ключей) объекта for-in. Цели и задачи, решаемые при помощи циклов. Задачи обработки наборов данных. Понятие сортировки, реализация пузырьковой сортировки. Понятие сложности алгоритма (концепция Big O) и оптимизации алгоритма. Замеры времени выполнения кода. Общие принципы работы с данными и WebAPI.
